# ✈ SkyFare — World's Cheapest Travel Booking Platform

> Built by **Pragyan Kumar Pattnaik**

A fully functional travel booking website with real-time flight prices, AI chatbot, multi-currency support, and Razorpay payments.

## 🌐 Live Site
**[https://skyfare.onrender.com](https://skyfare.onrender.com)**

## ✨ Features
- ✈ Real-time flight search (RapidAPI — Skyscanner data)
- 💱 Live currency conversion (150+ currencies)
- 🤖 24/7 AI travel chatbot (Groq AI — Llama 3)
- 💳 UPI + Card + NetBanking payments (Razorpay)
- 🌍 9 languages, fully responsive

## 🗂 Project Structure
```
skyfare/
├── public/
│   └── index.html      ← Complete website
├── server.js           ← Express server for Render
├── package.json        ← Node dependencies
├── render.yaml         ← Render config
└── .gitignore
```

## 🚀 Deploy on Render
1. Push this repo to GitHub
2. Go to render.com → New Web Service
3. Connect your GitHub repo
4. Build command: `npm install`
5. Start command: `npm start`
6. Deploy!
